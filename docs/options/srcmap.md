# -srcmap

Parameter: `filename`

Specifies a source map file.

If the source and destination are the same file and no merging is being performed, the -srcmap option is optional.

Example:

```
MMDatUtil.exe -srcmap "src.dat" -outmap "out.dat" -copysrc
```
