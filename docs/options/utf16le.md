# -utf16LE

Output map will be written using UTF16 little-endian output encoding.

By default, all UTF output formats will include a BOM unless you use the `-nobom` option.

This is the default format Manic Miners will use to save a map with any unicode data.

>Manic Miners cannot read UTF16BLE files without a BOM.
